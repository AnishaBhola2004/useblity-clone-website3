Array methods

  • toString

    • convert a array to a string

  • join()

    • join all the array elements using a seprator

  • pop()

    • remove last element from the array

  • push ()

    • adds new elements at the end of the array

  • shift ()

    • removes first element and returns it

  • unshift()

    • adds element in the beigning and returns new array length

  • delete ()

    • delete the element of array

  • concat

    • use to join one array to another

  • Sort ()

    • sort the array alphabetically

  • splice ()

    • use to add new items to array
